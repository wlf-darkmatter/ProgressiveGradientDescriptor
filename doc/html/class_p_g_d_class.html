calc_PGDFilter()函数，根据给定的圆周大小计算n个采样点的方向不变特征

参数
_src输入的矩阵
_dst返回的矩阵

返回

①通道数量转换

一律使用double类型，同时对边缘进行填充

②计算样本采样坐标偏移量

③计算每一个采样点的二次插值需要的参考权重（这里是N4方法）

④遍历全图
